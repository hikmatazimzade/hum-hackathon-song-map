We have the datasets that include songs from Ukraine and Estonia. The "Song Map" platform is built for daily users to explore
folk songs, learn and have fun with an interactive UI. The platform main features:

- Map (kind of like google map, but for the countries and regions are more emphasized) (Ukraine & Estonia separately) in a divided into regions way. Not the whole country map, but only the regions we have in our dataset should be displayed. The
  UI should be as minimalistic, elegant and simple as posssible so it's intuitive and fun for people to discover
  folk songs.
- Modern UI
- When user clicks on a region, the singers of that region appears
- When user clicks on a singer, the songs of that singer appear. Also, on the bottom, the recommendations of other singers and songs of that singer appear
  based on similarities
- When user clicks on song, the recommendations (both for singer and song) appear based on similarities
- Filter to drag through based on emotions (sad, happy from 0 to 1 spectrum)
- On the left side, we have themes (e.g: love, death, family relations) and the color density is decided. When user clicks on a theme the songs are shown belong to that theme appear. Also, for this part both (Estonian and Ukraine shown together). And, all the bars should be at the same length divided into these 3. And, based on how many songs each country has on these themes, the ratios differ. For example, Estonia has 33 songs on love, and Ukraine 500. Then, majority of it should be on Ukraine.
- Also, the collector data & collection time of the songs

- Extra song features like semantic complexity, line number etc.
